## Run Locally

Clone the project

```bash
  git clone https://github.com/mrityu2802/Affword-assignment.git
```

Go to the project directory

```bash
  cd frontend

  cd backend/src
```

Install dependencies

```bash
  npm install
```

Start the server

backend
```bash
  npm start
```

frontend
```bash
  npm run dev
```
